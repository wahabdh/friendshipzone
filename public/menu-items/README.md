# Menu Items Images Folder

This folder is where you should place your food item images.

## How to Add Your Images

1. **Create/Use the `menu-items` folder** - This folder already exists in `/public/menu-items/`

2. **Add your images** - Place your food item images here with the following names:
   - `veggie-burger.jpg` - For the Veggie Burger item
   - `chicken-wrap.jpg` - For the Spicy Chicken Wrap item
   - `pizza.jpg` - For the Margherita Pizza item
   - `paneer-tikka.jpg` - For the Paneer Tikka item
   - `biryani.jpg` - For the Biryani Special item

3. **Supported formats**: JPG, PNG, WebP, GIF

## When Adding New Menu Items

If you add new items through the admin panel, reference images with the path:
```
/menu-items/your-image-name.jpg
```

## Current Menu Items Expecting Images
